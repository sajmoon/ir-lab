{{Arena|namn            = Faxevallen|bild            = |bildtext        = |officiellt_namn = |tidigare_namn   = |smeknamn        = |placering       = [[Söderhamn]], {{Flaggbild2|Sverige}}|typ             = [[Bandy]]anläggning|ägare           = |underhåll       = |hemmalag        = [[Brobergs IF]]|kostnad         = |arkitekt        = |konstruktör     =|entreprenör     =|storlek         = |underlag        = |kapacitet       = |evenemang       = [[Bandy]]|webbplats       = |byggstart       = [[Juli]] [[Sportåret 1930#Bandy|1930]]|färdigställd    = |öppnade         = |renoverad       = |utbyggd         = |stängd          = |rivd            = |arenaranking    = }}'''Faxevallen''' är en [[bandy]]anläggning i [[Söderhamn]], [[Sverige]], som åren [[Sportåret 1930#Bandy|1930]]-[[Sportåret 1976#Bandy|1976]] var hemmaplan för [[Broberg/Söderhamn Bandy|Brobergs IF]].I [[juli]] [[Sportåret 1930#Bandy|1930]] sattes arbetet igång med idrottsplatsen Faxevallen. Med frivilliga insatser schaktade man och skottade grus. Faxevallen blev den stora vändningen för Brobergs IF.
